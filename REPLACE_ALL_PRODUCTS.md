# 🔄 החלפת כל המוצרים - מדריך

## מה זה עושה?

הפקודה `npm run replace:all` **מחליפה את כל המוצרים הקיימים** במוצרים חדשים מהלינקים שאתה מביא.

## ⚠️ אזהרה

**זה ימחק את כל המוצרים הקיימים ויחליף אותם בחדשים!**

## איך להשתמש?

### שלב 1: הבא 10 לינקים

שלח לי 10 לינקים של מוצרים אמיתיים מאמזון:

```
https://amzn.to/49OAnv4
https://www.amazon.com/dp/B08YZ5YF7M
https://amzn.to/XXXXX
...
(10 לינקים בסך הכל)
```

### שלב 2: אני אעדכן

אני אעשה:
1. אחלץ את ה-ASIN מכל לינק
2. אחליף את כל המוצרים ב-`lib/products-data.ts`
3. כל המוצרים החדשים יופיעו באתר

### שלב 3: עדכון ידני (אחר כך)

תצטרך לעדכן:
- ✅ שמות מוצרים (מאמזון)
- ✅ מחירים, דירוגים, ביקורות
- ✅ תמונות רלוונטיות
- ✅ mainProblem ו-targetUser

---

## דוגמה

```bash
npm run replace:all \
  "https://amzn.to/49OAnv4" \
  "https://www.amazon.com/dp/B08YZ5YF7M" \
  "https://amzn.to/XXXXX" \
  ... (10 לינקים)
```

## מה קורה אחרי?

1. כל המוצרים הישנים נמחקים
2. כל המוצרים החדשים מתווספים
3. האתר מתעדכן אוטומטית

**אבל** - תצטרך לעדכן:
- שמות מוצרים
- מחירים, דירוגים
- תמונות
- תיאורים

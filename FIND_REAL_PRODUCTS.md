# 🎯 איך למצוא מוצרים אמיתיים באמזון?

## ⚠️ הבעיה עם המערכת הנוכחית

המערכת `npm run scan:products` משתמשת ב-AI שמזהה מוצרים, אבל:
- ❌ **לא יכולה לוודא שהמוצרים אמיתיים**
- ❌ **ממציאה ASINs** שנראים אמיתיים אבל לא קיימים
- ❌ **לא יכולה לגשת לנתונים אמיתיים** בלי PA-API

**לכן צריך למצוא מוצרים אמיתיים בעצמך!**

---

## ✅ הפתרון: מצא מוצרים אמיתיים באמזון

### שלב 1: קבל אסטרטגיות חיפוש

```bash
npm run find:real
```

זה יציג לך:
- מילות מפתח לחיפוש באמזון
- קישורים לחיפוש באמזון
- מה לחפש במוצרים
- קריטריונים למוצרים טובים

---

### שלב 2: חפש באמזון

1. **פתח את הקישורים** שהמערכת נתנה לך
2. **עיין בתוצאות** באמזון
3. **בחר מוצרים** שעונים על הקריטריונים:
   - דירוג 4.0+ כוכבים
   - 500+ ביקורות
   - מחיר $15-$60

---

### שלב 3: קבל את ה-ASIN

כשאתה לוחץ על מוצר באמזון, ה-ASIN נמצא ב-URL:

```
https://www.amazon.com/dp/B08YZ5YF7M
                              ^^^^^^^^^^
                              זה ה-ASIN
```

**העתק את ה-ASIN** (10 תווים).

---

### שלב 4: בדוק שהמוצר אמיתי

```bash
npm run verify:product B08YZ5YF7M
```

או פשוט פתח את הקישור באמזון ובדוק שהמוצר קיים.

---

### שלב 5: הוסף לאתר

הוסף את המוצר ל-`ai/discoveryInput.json`:

```json
{
  "asin": "B08YZ5YF7M",
  "title": "Bamboo Drawer Organizer",
  "price": 24.99,
  "rating": 4.5,
  "reviews": 8500,
  "vertical": "home_kitchen",
  "baseAmazonUrl": "https://www.amazon.com/dp/B08YZ5YF7M",
  "trackingId": "aipicks20-20",
  "mainProblem": "messy kitchen drawers",
  "targetUser": "busy people who cook at home"
}
```

---

### שלב 6: ייבא לאתר

```bash
npm run import:discovery
```

## 💡 שימוש ב-Amazon SiteStripe (הכי קל!)

### מה זה SiteStripe?

SiteStripe הוא כלי של Amazon Associates שמאפשר לך:
- לראות את ה-ASIN של כל מוצר
- ליצור קישורי שותפים אוטומטית
- לקבל את כל המידע על המוצר

### איך להשתמש:

1. **התקן SiteStripe** מ-Amazon Associates:
   - לך ל: https://affiliate-program.amazon.com/
   - Tools → SiteStripe
   - התקן את הכלי

2. **עיין באמזון** וחפש מוצרים

3. **כשאתה על עמוד מוצר**, תראה את SiteStripe בראש העמוד

4. **לחץ על "Text Link"** - זה יצור קישור שותפים עם ה-ASIN

5. **העתק את ה-ASIN** מה-URL

6. **הוסף ל-`discoveryInput.json`**

## 📋 תהליך מלא - מציאת מוצרים אמיתיים

### אופציה א: עם SiteStripe (מומלץ)

```bash
# 1. קבל אסטרטגיות חיפוש
npm run find:real

# 2. פתח את הקישורים באמזון
# 3. השתמש ב-SiteStripe כדי לקבל ASINs
# 4. הוסף ל-discoveryInput.json
# 5. ייבא
npm run import:discovery
```

### אופציה ב: ידנית

```bash
# 1. קבל אסטרטגיות חיפוש
npm run find:real

# 2. פתח את הקישורים באמזון
# 3. לחץ על מוצר - ה-ASIN ב-URL
# 4. בדוק שהמוצר אמיתי
npm run verify:product [ASIN]

# 5. הוסף ל-discoveryInput.json
# 6. ייבא
npm run import:discovery
```

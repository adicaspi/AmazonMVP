# 🤖 Amazon Product Scanner & Recommender

## מה זה?

מערכת חכמה שסורקת מאות מוצרים של אמזון וממליצה על המוצרים הטובים ביותר לפרסום באתר.

## איך זה עובד?

### 1. סריקה אוטומטית של קטגוריות

המערכת סורקת קטגוריות שונות של אמזון:
- Kitchen Organization
- Home Storage
- Kitchen Tools
- Food Storage
- Home Decor & Organization

### 2. ניתוח AI

AI Agent מנתח כל מוצר לפי:
- **פוטנציאל המרה** - האם המוצר פותר בעיה אמיתית?
- **ביקורות ודירוגים** - 4.0+ כוכבים, 500+ ביקורות
- **מחיר** - $15-$60 (אידיאלי לרכישה אימפולסיבית)
- **בעיה-פתרון** - האם המוצר פותר בעיה יומיומית?
- **ערך ויזואלי** - האם יש ערך ברור?

### 3. דירוג והמלצות

כל מוצר מקבל ציון (0-100) והמלצה:
- **85+** - מוצר מעולה, מומלץ מאוד
- **75-84** - מוצר טוב, מומלץ
- **<75** - מוצר פחות מומלץ

## שימוש

### סריקה אוטומטית והמלצות

```bash
npm run scan:products
```

זה יסרוק מאות מוצרים ויצור קובץ `ai/recommended-products.json` עם ההמלצות.

### הוספת מוצר ידנית

אם יש לך ASIN ספציפי:

```bash
npm run add:product B08YZ5YF7M "Bamboo Drawer Organizer" 24.99 4.5 8500 kitchen "organization,storage,bamboo" "messy drawers" "busy people"
```

פרמטרים:
- `asin` - ASIN של המוצר (10 תווים)
- `title` - שם המוצר
- `price` - מחיר
- `rating` - דירוג (1-5)
- `reviews` - מספר ביקורות
- `room` - חדר (kitchen, living_room, etc.)
- `tags` - תגיות מופרדות בפסיקים
- `problem` - הבעיה שהמוצר פותר
- `user` - מי צריך את המוצר

### ייבוא מוצרים לאתר

לאחר שיש לך מוצרים ב-`discoveryInput.json`:

```bash
npm run import:discovery
```

זה יוסיף את כל המוצרים לאתר אוטומטית.

## תהליך המלצה

1. **סריקה** - `npm run scan:products`
2. **סקירה** - בדוק את `ai/recommended-products.json`
3. **אימות** - ודא שה-ASINs אמיתיים באמזון
4. **הוספה** - הוסף את המוצרים הטובים ל-`discoveryInput.json`
5. **ייבוא** - `npm run import:discovery`

## קריטריונים להמלצה

המערכת ממליצה על מוצרים עם:
- ✅ דירוג 4.0+ כוכבים
- ✅ 500+ ביקורות
- ✅ מחיר $15-$60
- ✅ פותר בעיה יומיומית
- ✅ ערך ברור ויזואלי
- ✅ פוטנציאל המרה גבוה

## דוגמה

```bash
# 1. סרוק מוצרים
npm run scan:products

# 2. בדוק המלצות
cat ai/recommended-products.json

# 3. הוסף מוצר ספציפי (אופציונלי)
npm run add:product B08YZ5YF7M "Bamboo Drawer" 24.99 4.5 8500 kitchen "organization" "messy drawers" "busy people"

# 4. ייבא לאתר
npm run import:discovery
```

# ✅ איך לוודא שמוצר אמיתי באמזון?

## ⚠️ למה המערכת לא יכולה לוודא?

**בלי PA-API (Product Advertising API) של אמזון, אין דרך אוטומטית לוודא שהמוצרים אמיתיים.**

המערכת משתמשת ב-AI שמזהה מוצרים פופולריים, אבל:
- ❌ לא יכולה לבדוק אם ASIN אמיתי
- ❌ לא יכולה לוודא שהמוצר קיים
- ❌ לא יכולה לבדוק מחיר, דירוג, ביקורות אמיתיים

**לכן צריך לבדוק ידנית!**

---

## 🔍 איך לבדוק אם מוצר אמיתי?

### שלב 1: בדוק מוצר בודד

```bash
npm run verify:product B08YZ5YF7M
```

זה יציג לך:
- את הקישור לאמזון
- הוראות לבדיקה ידנית

### שלב 2: פתח את הקישור באמזון

הקישור יהיה:
```
https://www.amazon.com/dp/B08YZ5YF7M
```

### שלב 3: בדוק אם המוצר אמיתי

✅ **המוצר אמיתי אם:**
- העמוד נטען (לא "Page Not Found")
- יש תמונה של המוצר
- יש שם, מחיר, דירוג, ביקורות
- אפשר לקנות את המוצר

❌ **המוצר לא אמיתי אם:**
- מופיע "Page Not Found" או "Sorry, we couldn't find that page"
- אין תמונה או מידע
- העמוד לא נטען

---

## 📋 בדיקת כל המוצרים המומלצים

```bash
npm run verify:all
```

זה יציג לך רשימה של כל המוצרים המומלצים עם קישורים לבדיקה.

---

## 🛠️ תהליך מלא - איך לעבוד עם מוצרים

### 1. סרוק מוצרים
```bash
npm run scan:products
```

### 2. בדוק את ההמלצות
```bash
npm run verify:all
```

### 3. פתח כל קישור ובדוק ידנית

לכל מוצר:
1. פתח את הקישור באמזון
2. ודא שהמוצר קיים
3. בדוק שהמחיר, הדירוג, והביקורות נכונים
4. רק אם הכל נכון - הוסף ל-`discoveryInput.json`

### 4. הוסף רק מוצרים מאומתים

**רק מוצרים שבדקת ידנית ואתה יודע שהם אמיתיים!**

### 5. ייבא לאתר
```bash
npm run import:discovery
```

## 💡 טיפים לבדיקה

✅ **תמיד בדוק לפני הוספה** - אל תסמוך על AI בלבד
